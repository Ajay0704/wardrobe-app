## iOS (Capacitor)

Native shell for personal iPhone testing (WebView → production URL). See [[iOS Capacitor]] for Xcode signing, free Apple ID limits (~7-day reinstall), and local debug.

```bash
npm run cap:sync
npm run cap:open:ios
```
